# Factor Dopamina

Factor Dopamina es una plataforma web de educacion y entretenimiento digital enfocada en cursos, contenido formativo y gestion administrativa de programas desde un panel interno.

El proyecto esta construido como una aplicacaion full-stack con frontend en React, backend en FastAPI y base de datos en MongoDB Atlas.

---

## Tecnologias principales

### Frontend

- React
- react Router
- Tailwind CSS
- Axios
- CRACO
- Lucid React

### Backend

- Python 3.11
- FastAPI
- Uvicorn
- Pydantic
- PyMongo / Motor
- JWT
- Bcrypt
- Python Dotenv

### Base de datos

- MongoDB Atlas
